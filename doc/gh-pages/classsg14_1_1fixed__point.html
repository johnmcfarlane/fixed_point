Detailed Description

template<class ReprType = int, int Exponent = 0>
class sg14::fixed_point< ReprType, Exponent >

literal arithmetic type which approximates a real number

Parameters
ReprTypethe underlying type used to represent the value
Exponentthe value by which to scale the integer value in order to get the real value
Examples

To store the value, -2.75, in a 1-byte variable with 3 integer and 4 fractional digits:

constexpr auto n = fixed_point<char, -3>{-2.75};
static_assert(n == -2.75, "fixed-point type was unable to store the value");
